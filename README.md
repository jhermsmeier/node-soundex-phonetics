## Install via [npm](https://npmjs.com)

```sh
$ npm install soundex-phonetics
```

## Usage

```js
var soundex = require( 'soundex-phonetics' )
```

```js
if( soundex( 'Ekzampul' ) === soundex( 'Example' ) ) {
  // It sounds similar!
}
```
